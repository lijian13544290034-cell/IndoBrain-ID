# Product Philosophy

Version: v1.0

## Mission

Help Chinese users live and work in Indonesia with less friction.

## Core Idea

Solve a real-life problem first. Language learning happens naturally while solving it.

## Product Principles

1. Real situations are more valuable than textbook examples.
2. Useful communication is more important than perfect grammar.
3. Indonesian is the primary working language; Chinese is supportive context.
4. Experiences should capture repeatable solutions, not only information.
5. Every interaction should make the next user’s task easier.

## Experience Loop

Event → Task → Dialogue → Experience → Feedback → Learning

## Design Direction

The product should feel practical, calm, warm, and easy to use every day.
