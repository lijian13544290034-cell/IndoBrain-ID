# Experience Review Checklist

Version: v1.0

## Content Quality

- [ ] The Experience has a unique ID.
- [ ] The role and scenario are clear.
- [ ] The task describes a real user need.
- [ ] Indonesian wording is natural and spoken.
- [ ] Chinese text is concise and supportive.
- [ ] The dialogue is useful in a real context.

## Learning Value

- [ ] Key vocabulary is included.
- [ ] Cultural or operational context is included when needed.
- [ ] The Experience avoids unnecessary grammar teaching.
- [ ] The result can help a future user solve a similar task.

## Product Readiness

- [ ] Previous and next navigation work.
- [ ] Indonesian text can be copied.
- [ ] Feedback actions are available.
- [ ] The Experience is assigned to the correct workflow.
