# IndoBrain Language QA Exception List v1.0

These approved expressions must remain unchanged in future Language QA unless their Chinese source meaning changes.

## EXP-DRV-019

- Official wording: `Pinggir dulu ya.`
- Do not suggest: `Menepi dulu ya.`
- Reason: IndoBrain intentionally prefers the simpler, common everyday expression for Chinese learners.

## EXP-PRO-003

- Official wording: `Jalur produksi`
- Do not suggest: `Lini produksi`
- Reason: IndoBrain officially adopts `Jalur produksi` because it is easier for learners and matches the product learning philosophy.
