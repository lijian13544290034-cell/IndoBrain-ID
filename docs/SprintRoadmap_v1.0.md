# Sprint Roadmap

Version: v1.0

## Current Foundation

- Experience browsing for Driver, ART, and Factory Manager.
- Anonymous session support.
- AI provider adapter and unconfigured fallback state.
- Lightweight Supabase event, feedback, and conversation integration.
- Initial Pattern and Variable demonstration.

## Next Priorities

1. Complete and review canonical Driver and ART Experience source content.
2. Run the first real-user test with anonymous sessions.
3. Validate AI provider quality, reliability, and cost.
4. Review feedback and learning progress data.
5. Improve Experience templates from real user conversations.

## Later Work

- Progressive localization behavior.
- Expanded factory roles.
- Experience review workflow.
- Pattern library growth.
- Production deployment and monitoring.

## Out of Scope

- Authentication and payments.
- Social features.
- Complex administration tools.
