# IndoBrain Documentation

## Project

IndoBrain is a practical Indonesia life and work assistant for Chinese users.

## Architecture

The project uses Next.js, an AI Provider Adapter, Supabase, and Experience-based modules.

## Folder Structure

- `app/` — Application routes and API handlers.
- `components/` — Reusable user interface components.
- `experience/` — Experience source content.
- `lib/` — Shared application logic.
- `docs/` — Project documentation.

## Development Workflow

Define the sprint scope, implement the smallest useful change, verify the build, and record user-visible changes.
